Генератор utm-меток

Как пользоваться генератором?

Генератор ссылок поможет вам подготовить ссылку на продукт Тинькофф Бизнес c UTM-меткой для использования внутри статьи. Для этого:

1. Заполните поля:

Ссылка, куда добавить utm-метку: ссылка на продукт Тинькофф Бизнес. Обычно выглядит так: www.tinkoff.ru/business/название_продукта

Ссылка на статью или ее слаг: можно ввести либо полную ссылку на статью (например, https://secrets.tinkoff.ru/lichnyj-opyt/kak-sozdat-soobshchestvo/), либо только последнюю часть адреса (kak-sozdat-soobshchestvo)

Рубрика статьи: выберите рубрику статьи из выпадающего списка. Доступен встроенный поиск.

Расположение ссылки: если ссылка будет находиться в выноске, выберите первый вариант, если в тексте статьи, то второй.

Нестандартный продукт: если продуктовая ссылка начинается не с www.tinkoff.ru, то поставьте галочку Нестандартный продукт и выберите название продукта из выпадающего списка.

2. Нажмите кнопку Сгенерировать ссылку

3. Скопируйте готовую ссылку из поля Результат
