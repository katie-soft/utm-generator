Генератор utm-меток

Как пользоваться генератором?

Чтобы сгенерировать UTM-метку для ссылки, сделайте следующее:

  1. Вставьте ссылку на продукт Тинькофф в поле Ссылка на продукт. Например, http://www.tinkoff.ru/business/название_продукта
  2. Добавьте во второе поле ссылку на статью или слаг — именно из этой статьи будет ссылка на продукт. Например, https://secrets.tinkoff.ru/lichnyj-opyt/kak-sozdat-soobshchestvo/, либо только последнюю часть адреса — слаг (kak-sozdat-soobshchestvo).
  3. Выберите рубрику статьи. Например, Личный опыт.
  4. Расположение ссылки: если ссылка на продукт будет находиться на полях (выноске), то доступен вариант В выноске, если просто в тексте — выберите В тексте.
  5. Если все заполнено корректно, нажмите Сгенерировать ссылку.
  6. Если нужно сделать другую ссылку, очистите поля с помощью кнопки Очистить форму.
